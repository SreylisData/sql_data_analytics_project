# SQL Exploratory Data Analysis (EDA)
## **Overview**

This project is a collection of SQL scripts designed to analyze customer behavior, product performance, and sales trends. The goal is to transform raw transactional data into actionable insights that support data-driven decision-making.

The repository demonstrates how SQL can be used not just for querying data, but for building structured, repeatable analytics workflows.

---
## **Objectives**

* Explore and understand raw data structures
* Develop key business metrics (KPIs)
* Segment customers and products based on behavior and performance
* Analyze trends over time
* Deliver insights that support business decisions

---
## **Key Insights**

* A small segment of customers contributes disproportionately to total revenue (VIP segment)
* Product performance varies significantly, with some high-volume products generating lower profit margins
* Customer purchasing patterns reveal differences in recency and frequency
* Sales trends highlight periods of growth and stagnation
* Segmentation enables clearer identification of high- and low-performing groups

---
## **Analytical Approach**

This project applies SQL to:

* Perform data exploration (schema, distributions, time ranges)
* Aggregate key metrics (sales, orders, customers)
* Segment data using conditional logic
* Analyze trends using time-based queries
* Rank and compare performance across dimensions

---
## **Reports** 

**Customer Analytics**

Provides insights into customer behavior and value:

* Customer segmentation (VIP, Regular, New)
* Total orders, sales, and quantity purchased
* Customer lifespan and recency
* Average Order Value (AOV)
* Monthly spending patterns

**Product Performance**

Evaluates how products perform across key metrics:

* Product segmentation (High, Mid, Low performers)
* Sales and order volume
* Customer reach per product
* Product lifespan
* Average Order Revenue (AOR)
